Ask A Manager – Salary Analytics 2021

Fuente: Ask A Manager Salary Survey 2021
Herramienta: Looker Studio

Descripción del proyecto

Este sitio documenta el análisis salarial basado en la encuesta Ask A Manager Salary Survey 2021. El proyecto integra procesos de limpieza de datos, modelado analítico, normalización geográfica y conversión monetaria para permitir comparaciones homogéneas entre industrias y países.

El objetivo principal es analizar la relación entre industria, experiencia profesional y compensación total anual, expresando todos los valores monetarios en pesos colombianos (COP).

Dashboard

El dashboard fue desarrollado en Looker Studio y se encuentra conectado al modelo de datos previamente transformado.

Conversión de moneda

  • USD, GBP, CAD, EUR, AUD/NZD → COP
  • CHF, JPY, SEK, ZAR, HKD → COP
  • Para “Other”, se validó el campo de especificación manual

Notas metodológicas

  • Las variables ordinales conservan su jerarquía lógica
  • Las métricas monetarias se estandarizaron en COP
  • Los campos geográficos fueron normalizados

Paso a paso del modelado

  1. Carga de datos en Looker Studio
  2. Limpieza y normalización de texto
  3. Conversión monetaria
  4. Creación de métricas agregadas
  5. Construcción de visualizaciones
  6. Actualización mediante reemplazo del archivo fuente

Conclusión

El modelo permite analizar de forma clara la relación entre industria, experiencia, nivel educativo y salario, garantizando comparabilidad internacional mediante la conversión a pesos colombianos.
